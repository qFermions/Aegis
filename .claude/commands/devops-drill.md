# /devops-drill

Deliver ONE practical DevOps drill per invocation, using **Variant D** of `docs/command-output-standard.md`:

```
## Goal → ## Aha moment → ## Lab (numbered, hands-on) → ## Verify
→ ## Common mistake → ## Career upgrade → ## Stretch task (optional)
```

**Topic selection:**
- Operator's argument wins; otherwise pick what connects to recent real work (e.g., the pre-commit hook → Git hooks lab; Koinon promotion flow → branch/PR workflow lab).
- Good pool: Git branch workflow & PRs, CI/CD pipeline basics (GitHub Actions), YAML, Dockerfile + Docker Compose, environment variables & secrets management, monitoring/logging, rollback strategy, Terraform basics, infrastructure-as-code for the Aegis repo itself.

**Rules:**
- Hands-on in this repo or a scratch folder — never against production systems without the standard admin gate.
- Placeholders only; no real credentials in any example (Koinon SR-1).
- One drill, phone-screen readable, clean stop with a next action.
